# CampusPe Gen AI Chat UI

## Description
[cite_start]This project is a modern, responsive chat user interface built for the CampusPe Gen AI Assignment[cite: 7]. [cite_start]It simulates a ChatGPT or Claude-style chat experience using front-end technologies without a backend[cite: 9, 10]. [cite_start]The interface focuses on clean design, smooth animations, and intuitive user experience[cite: 14, 18, 120].

## Technologies Used
* [cite_start]HTML5 [cite: 20]
* [cite_start]CSS3 (with Custom Properties/Variables) [cite: 21, 56]
* [cite_start]JavaScript (ES6) [cite: 22]
* [cite_start]jQuery (v3.7.0) [cite: 23]
* [cite_start]Bootstrap 5 [cite: 3, 10]
* [cite_start]Font Awesome Icons [cite: 42]
* [cite_start]Google Fonts (Inter) [cite: 42]

## Features Implemented
* [cite_start]**Responsive Design:** Works seamlessly across desktop, tablet, and mobile devices[cite: 17, 109, 123].
* [cite_start]**Semantic HTML:** Structured using proper HTML5 semantic tags[cite: 40].
* [cite_start]**Dynamic Input Handling:** Textarea auto-resizes as the user types[cite: 86]. [cite_start]Send button enables/disables based on input[cite: 83].
* [cite_start]**Keyboard Shortcuts:** `Enter` to send, `Shift+Enter` for a new line[cite: 84].
* [cite_start]**Mock AI Responses:** Simulates a 1.5-second processing delay before the assistant replies[cite: 90].
* [cite_start]**Auto-Scrolling:** Chat automatically scrolls to the bottom when new messages are added[cite: 81].
* [cite_start]**Mobile Sidebar:** Off-canvas sidebar with a dark overlay for mobile navigation[cite: 103, 104].

### Bonus Features Included
* [cite_start]**Dark Mode Toggle (4 pts):** Smooth transition between light and dark themes using a floating action button[cite: 111].
* [cite_start]**Typing Animation (4 pts):** The AI response is typed out letter-by-letter, mimicking real-time generation[cite: 111].

## How to Run the Project
1. [cite_start]Extract the `YourName_ChatUI.zip` folder[cite: 4, 132].
2. [cite_start]Open the folder and locate the `index.html` file[cite: 136].
3. [cite_start]Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari recommended)[cite: 25, 164].
4. [cite_start]No local server or backend installation is required[cite: 10].
